# 🛒 E-Commerce Data Intelligence Pipeline

## 📝 Resumo do Projeto
Este projeto consiste no desenvolvimento de um **Pipeline de Dados de ponta a ponta (End-to-End)** focado em inteligência de negócio e análise de cenários para e-commerce, utilizando a base de dados pública e real da **Olist** (Brazilian E-Commerce Public Dataset). 

O objetivo principal é demonstrar maturidade técnica na estruturação de dados relacionais, análise exploratória avançada e entrega de indicadores estratégicos (KPIs) para tomadas de decisão executiva.

## 🛠️ Stack Tecnológica (Desenvolvimento Progressivo)
* **Banco de Dados & ETL:** SQL Server / T-SQL
* **Análise Exploratória:** Python (Jupyter Notebook / Pandas)
* **Data Visualization:** Power BI (DAX / Modelagem Dimensional)
